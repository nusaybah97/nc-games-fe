# NC GAMES

Welcome to NC Games where you can view information related to various different board games, read reviews and leave comments.

The app can be viewed here [NCGames](https://nusaybah-games.netlify.app/)

## Local Setup

### Forking and Cloning

In order to have your own versions of the repository you will need to fork it. You can then clone it down to your machine locally. 

```
git clone <paste the link to your repository here>
```

### Dependencies

Run the following command to install all the required dependencies:

```
npm install
```

### Starting up

To start the app on your machine run 
```
npm start
```
and then open your browser and go to http://localhost:3000 to view the app and any changes that you wish to make.
